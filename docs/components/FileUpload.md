`FileUpload`
============



Props
-----

### `accept`

File type acceptance.

- type: `string`


### `allowDrop`

- type: `bool`
- default value: `false`


### `multiple`

Whether to accept multiple files.

- type: `bool`
- default value: `false`


### `onChange`

Callback to be executed when there is a change in the value of the field.

- type: `func`
