`FieldImage`
============



Props
-----

### `config`

Image specific config.

- type: `object`


### `onChange`

Callback to be executed when there is a change in the value of the field.

- type: `func`


### `onError`

Callback to be executed when there is a new validation error in the field.

- type: `func`


### `schema`

The field schema.

- type: `object`


### `value`

The field value.

- type: `object`
