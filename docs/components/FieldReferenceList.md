`FieldReferenceList`
====================

Component for rendering API fields of type String on a list view.

Props
-----

### `collection`

The name of the collection being edited, as per the URL.

- type: `string`


### `currentApi`

The schema of the API being used.

- type: `object`


### `schema`

The field schema.

- type: `object`


### `value`

The field value.

- type: `string`
