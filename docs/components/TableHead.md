`TableHead`
===========

A table head.

Props
-----

### `allSelected`

Whether all the rows in the table are currently selected.

- type: `bool`
- default value: `false`


### `children`

The contents of the table head.

- type: `node`


### `selectable`

Whether rows are selectable. When `true`, check boxes will automatically be added to the table head and to each row.

- type: `bool`
- default value: `false`
