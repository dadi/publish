`RichEditor`
============

A rich text editor.

Props
-----

### `children`

The content of the editor.

- type: `node`


### `format`

The format used for input and output.

- type: `enum('html'|'markdown')`


### `onChange`

A callback function that is fired whenever the content changes.

- type: `func`


### `value`

The initial value of the editor.

- type: `string`
