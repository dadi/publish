`Nav`
=====

The main navigation component.

Props
-----

### `currentCollection`

- type: `object`


### `items`

Grouped list of navigation elements to render.

- type: `array`
- default value: `[]`


### `mobile`

Whether to render the navigation in mobile mode,
with a collapsible drawer controlled by a hamburger button.

- type: `bool`
- default value: `false`
