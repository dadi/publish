`IconArrow`
===========

An arrow icon.

Props
-----

### `class`

A list of classes to be added to the icon element.

- type: string
- default value: `null`


### `direction`

Colour accent.

- type: enum('down'|'left'|'right'|'up')
- default value: `'up'`


### `height`

The height of the arrow.

- type: number
- default value: `10`


### `width`

The width of the arrow.

- type: number
- default value: `10`
