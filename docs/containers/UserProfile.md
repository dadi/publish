`UserProfile`
=============

The interface for editing a user profile.

Props
-----

### `section`

The current active section (if any).

- type: `string`


### `sections`

The current active section (if any).

- type: `array`
- default value: `[
  {
    slug: 'account',
    value: 'Account'
  },
  {
    slug: 'settings',
    value: 'Settings'
  },
  {
    slug: 'security',
    value: 'Security'
  }
]`
